The posts in this section cover a broad range of activities, including the development of my curriculum vitae, preparation for my internship and bachelor thesis, participation in hackathons and Tech&Meet events, academic and self-directed projects, and continuous reflection on learning and professional growth.

Each post focuses not only on what was done, but also on what was learned, why it matters, and how it contributes to my development as a cybersecurity professional. Where relevant, reflections reference real-world tools, industry practices, and transferable professional skills.

Together, these reflections demonstrate continuous learning, professional awareness, and alignment with the expectations of a junior cybersecurity professional.

How this website was created

This website was created as an e-portfolio for the course Professional Networking at Howest University of Applied Sciences. The goal is to present academic work, professional activities, and extracurricular involvement in a structured and accessible way, while also serving as a professional portfolio for job applications.

Platform & Hosting

Hosted using GitHub Pages. The website was built manually using HTML, CSS, and JavaScript, without frameworks or static site generators, allowing full control over structure, styling, and behavior. HTTPS is enforced by default.

Content & Maintenance

All content and source code are maintained in a GitHub repository using Git. Changes are tracked through commits and deployed by updating the repository, supporting traceability and continuous improvement throughout the year.

Why this file exists

This website follows a traditional multi-file structure using separate HTML, CSS, JavaScript, and image files. JavaScript is used to enhance navigation and content interaction, allowing sections such as detailed views to be displayed dynamically without requiring additional page reloads.
